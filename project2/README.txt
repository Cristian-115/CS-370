
In order to  compile the following program a virtual inviorment is needed to install the the following packages bitarray and mmh3.
this can be done using the following steps:

Create a virtual environment in your current directory for a project with the command: virtualenv my_project

Start the virtual environment by activating with the command: source my_project/bin/activate

once the venv is set up you can now PIP install bitarray and mmh3 

Finally to run the program type python {filename}

type "deactivate" to stop the enviorment
